# migrationChapter
“CorRecD” analysis is divided in three parts: segmentation of the protrusion, generation of edge velocity and recruitment maps, cross-correlation calculation. 

This repository contains:
- in dataTest: some data and links to data for testing CorRecD
- imageJ: source code for the Macro and the Plugin (part 1 and 2)
- MATLAB: source code for Correlation Heat Map exe
- ReadytoUse: Macro for step1, plugin ready to be installed  (.jar) for part 2, and link to download the installer for Correlation Heat Map.exe for part 3. 
-VideoTutorials: Videos of part 1, part 2 and part 3 with example datasets.
